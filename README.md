# OculusLinkCrutch
Utility for fix Oculus Link and AirLink Freezes on Windows 11 - Enable/Disalbe Toggle Console Windows Visibility Mode and Service Priority with One Click

```
[settings]
OculusDebugToolFullPath=C:\Program Files\Oculus\Support\oculus-diagnostics\OculusDebugTool.exe
OVRServerMaxPriorety=2
```
OVRServerMaxPriorety: 0=Normal, 1=High, 2=Real Time
